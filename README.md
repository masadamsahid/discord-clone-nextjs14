# Discord Clone

Techs:
- React (Next.js)
- Shadcn-UI
- Tailwind CSS
- Prisma
- MySQL
- Uploadthing (storing images and PDFs)
- Socket.io (for realtime chat)
- Livekit (for audio & video call)
